tabele izabrane fudbalske lige. Rx vrši prikupljanje i osnovno mapiranje podataka o timovima i

server prima HTTP GET zahteve i prevodi ih u poruke aktorima. Prikazati poziciju na tabeli, broj
bodova i izračunati procenat uspešnosti.
